Contact

Soupault's maintainer is Daniil Baturin.

If you...

  • have a question about soupault
  • use soupault and want to be included in the user list
  • made a plugin or a script that works with soupault and want it included in the plugin directory

feel free to send a message to the mailing list.

Reporting bugs and submitting patches

If you have found a bug, the best place to report it is github.com/dmbaturin/soupault/issues. Feature requests are welcome, too.

A mirror of the source code is kept at git.sr.ht/~dmbaturin/soupault. If you don't want to use Github, you can send bug reports and patches to the mailing list instead.
