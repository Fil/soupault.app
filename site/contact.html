Contact

Soupault's maintainer is Daniil Baturin.

If you have found a bug, the best place to report it is github.com/dmbaturin/soupault/issues. Feature requests are welcome, too.

If you are using soupault and want your website added to the user list, or you just want to contact the maintainer directly, send an email to daniil+soupault@[maintainer's last name].org. If you made a plugin, or a script that works with soupault, I'm also happy to include it in the plugin directory.

A mirror of the source code is kept at git.sr.ht/~dmbaturin/soupault. If you don't want to use Github, I don't mind emailed patches.
